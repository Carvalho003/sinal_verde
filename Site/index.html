Sinal Verde

controle de trânsito inteligente

Nosso objetivo é melhorar o fluxo de carros nas ruas diminuindo estresse e acidentes

O QUE OFERECEMOS?

Uma solução IOT, utilizando de sensores de distância, para monitorar fluxo
de carros em trânsito, acoplado a um sistema web com uma dashboard
que possibilita a visualização tempo real do fluxo e os dados captados.
SENSORES

Nossa solução se baseia no uso do sensor HC-SR04, também conhecido como sensor ultrassônico, que capta a distância de um objeto próximo, esses sensores serão instalados em pontos estratégicos das vias para medir a distância de veículos e pedestres, para que seja possível fazer uma análise.

ADMINISTRAÇÃO

A aplicação web, conta com um sistema de administração de usuários, onde é possível adicionar novos colaboradores em sua empresa, tornando fácil o controle de acessos.

DASHBOARD

Nossa Dashboard proporciona aos usuários informações importantes para tomada de decisão, Informações chave e gráficos para facilitar a visualização das informações monitoradas pelos sensores ao decorrer do tempo estão presentes.
